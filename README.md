
# Blog

Blog Application with the functionalities :
- Login / Signup :lock:
- View Articles / View Single Articles
- Add Article :pencil2:
- View Authors

Each Article have a :
- Authors :technologist:
- Date of Creation 
- Tag (Field of Interest i.e: Tech , medecin...)
